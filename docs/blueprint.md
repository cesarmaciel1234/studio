# **App Name**: RutaRápida Driver

## Core Features:

- Driver Profile & Vehicle Management: Allow drivers to create and manage their personal details, contact information, vehicle type, and availability status.
- Order Assignment & Listing: Display a comprehensive list of pending, assigned, and completed delivery orders with essential details such as pickup/delivery locations and package information.
- Real-time Map & Navigation: Provide an interactive map view showing the driver's current location, pickup points, and delivery destinations, with visual guidance.
- AI-Powered Route Optimization Tool: A generative AI tool that analyzes multiple delivery stops, traffic conditions, and time windows to suggest the most efficient and time-saving route for the driver.
- Order Status Updates: Enable drivers to quickly update the status of each order (e.g., 'Picked Up', 'In Transit', 'Delivered', 'Problem') with timestamps and proofs like photos or signatures.
- In-App Alerts & Notifications: Receive instant alerts for new order assignments, order changes, cancellations, or important messages from the dispatcher or customer.
- Communication Hub: Facilitate direct chat or call functionality between the driver and customers, or between the driver and central dispatch support.

## Style Guidelines:

- Primary color: Deep Midnight Blue (#0F254F). This dark, authoritative blue reflects professionalism and is used for active states and key navigational elements, inspired by the active 'RUTA' tab.
- Background color: Cool Off-White (#F4F5F7). A very light, subtly blue-tinted background, reminiscent of the map background, ensures content readability and a clean interface.
- Accent color: Vibrant Verdant Green (#45C64D). A lively green hue for live indicators, notifications, and positive action buttons, directly inspired by the 'LIVE' status indicator in the map view.
- Body and headline font: 'Inter' (sans-serif). A modern, objective, and highly legible font choice suitable for clear information display, consistent with the functional nature of a delivery application.
- Use clear, minimalist line icons with a consistent style, as seen in the bottom navigation. Icons should be easily understandable and communicate their function instantly.
- Employ a modular and intuitive layout with card-like components for information organization. Key interactive elements, such as navigation, should be easily accessible, preferably via a bottom navigation bar.
- Implement subtle and smooth animations for state changes, map transitions, and menu interactions to enhance user experience without causing distraction or delays.
